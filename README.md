# Project Sphinx

## Project Vision
Sphinx is a comprehensive personal finance and productivity ecosystem. While its ultimate goal is to become a secure, cross-platform commercial application featuring AI analysis, OCR receipt parsing, and social sharing capabilities, its foundational purpose is to serve as a robust, hands-on learning vehicle for mastering software engineering. 

## Current State
* **Phase 1 (MVP) Development:** Building the core financial logic and mathematics using a Java CLI interface.

## Technical Stack
* **Current:** Java (Core)
* **Future:** Cross-platform mobile/web frameworks, Cloud Sync, Encryption standards, ML/AI integrations, OCR tools.

## Roadmap / Next Steps
* **Phase 1: Core Engine (MVP):** Fully replace manual spreadsheet tracking with a reliable, tested CLI financial calculator.
* **Phase 2: Data & Security:** Implement local data persistence, robust data encryption for sensitive financial records, and modular architecture.
* **Phase 3: Cross-Platform & Sync:** Expand to Web and Mobile platforms with offline-first capabilities and network-based cloud synchronization.
* **Phase 4: Advanced Modules:** Integrate AI agents for financial analysis, OCR for automated expense entry, habit trackers, and social sharing (budgets/goals).
